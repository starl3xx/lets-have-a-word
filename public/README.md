# Public Assets

This folder contains static assets that are served directly by Next.js.

Files in this directory can be referenced in your code starting from the base URL `/`.

For example:
- `public/logo.png` can be referenced as `/logo.png`
- `public/images/avatar.jpg` can be referenced as `/images/avatar.jpg`

## Common Asset Types

- Images (PNG, JPG, SVG, etc.)
- Fonts
- Icons
- Favicons
- robots.txt
- sitemap.xml
